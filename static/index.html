Cloud Dashboard Service

Interact with real-time country, weather, and currency data through clean RESTful endpoints.

Home API Docs Status

GET Endpoints

  • /dashboard/v1/registrations/
  • /dashboard/v1/dashboards/{id}
  • /dashboard/v1/notifications/
  • /dashboard/v1/status/

Config Examples

POST – Register

{
  "country": "Norway",
  "isoCode": "NO",
  "features": {
    "temperature": true,
    "coordinates": true,
    "capital": true,
    "targetCurrencies": ["EUR", "USD"]
  }
}

PATCH – Update

{
  "features": {
    "capital": false,
    "temperature": false
  }
}

Webhooks

Register:

{
  "event": "LOW_TEMP",
  "country": "NO",
  "url": "https://webhook.site/your-url"
}

Supported Events:

  • REGISTER
  • CHANGE
  • PATCH
  • DELETE
  • INVOKE
  • LOW_TEMP
